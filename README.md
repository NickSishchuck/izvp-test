# Універсальний воркфлоу
1. Клонувати репозиторій (1 раз)
```
  git clone https://github.com/NickSishchuck/izvp-test.git
```
  Перейти в репозиторій 
2. Перейти у свою гілку
```
  git checkout dev/{yourname}
```
3. Зробити зміни
```
  git add .
  git commit -m "Add your description here"
  git push origin dev/{yourname}
```
4. Створити PR на Github та чекати апрува від ліда
